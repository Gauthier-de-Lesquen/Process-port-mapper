# Process port mapper

> This Batch file looks for all the ports (like HTTP or SMTP for instance) opened and indicates the PID and file's name of the application.

## Why to use it?

This file is a way to check if there is any malware in the PC scanned.

## How to use it?

⚠️ **To use it, you must be on a Windows device.**
* Create a new file (using notepad if you don't have any IDE), paste the code and be sure that the file has the ```.bat``` extension.
* the file's icon iw then a logo with gears - that's normal.
* close all the apps opened to not deceive the scanner.
* double-click on the file and it will run by itself.
